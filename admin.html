🔌 API 接口说明

配置 banApi.mode = 'remote' 后，可通过以下 API 管理封禁列表：

GET /api/bans — 获取封禁列表

POST /api/bans — 添加封禁

DELETE /api/bans/:id — 删除封禁

PATCH /api/bans/:id/unban — 解封玩家

所有 API 请求需在 Header 中携带 Authorization: Bearer {apiKey} 进行鉴权。

★ 管理员密码用于后台登录鉴权，API 密钥用于接口鉴权，两者在 config.js 中独立配置。

📖 如何获取 SMTP 授权码？

1️⃣ 登录 QQ 邮箱 → 设置 → 账户

2️⃣ 找到「POP3/IMAP/SMTP 服务」→ 开启 SMTP 服务

3️⃣ 点击「生成授权码」→ 按提示发送短信后即可获取

4️⃣ 将生成的授权码填入上方的「SMTP 授权码」字段

⚠️ 授权码与 QQ 邮箱密码不同，请勿混淆！授权码可在 QQ 邮箱设置中随时重新生成。
